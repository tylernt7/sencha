  • What is the core difference between the @Bean and @Component annotations in Spring?
    • @Bean: Used on methods in @Configuration classes to manually create and configure a Spring bean.
    @Component: Used on classes to mark them for automatic detection and registration as beans via classpath scanning.
    @Bean is method-level and explicit; @Component is class-level and implicit via scanning.
