Connection String

Npgsql (.NET data provider for PostgreSQL)

Parameter Description Default
Host Specifies the host name. Required
Port The TCP port of the PostgreSQL server. 5432
Database The PostgreSQL database to connect to. Same as Username
Username (User ID) The username to connect with. PGUSER
Password The password to connect with. PGPASSWORD
Trust Server Certificate Whether to trust the server certificate without validating it false
Persist Security Info Gets or sets a Boolean value that indicates if security-sensitive information, such as the password, is not returned as part of the connection if the connection is open or has ever been in an open state. false
Include Error Detail When enabled, PostgreSQL error and notice details are included on PostgresException.Detail and PostgresNotice.Detail. false
Maximum Pool Size The maximum connection pool size (the default in the PostgreSQL server is 100). 100 since 3.1, 20 previously
Connection Idle Lifetime The time (in seconds) to wait before closing idle connections in the pool if the count of all connections exceeds Minimum Pool Size. 300
Connection Lifetime The total maximum lifetime of connections (in seconds) 3600 (1 hour), in Npgsql 8.0 and before - 0 (disabled)
Timeout The time to wait (in seconds) while trying to establish a connection before terminating the attempt and generating an error. 15
Command Timeout The time to wait (in seconds) while trying to execute a command before terminating the attempt and generating an error. Set to zero for infinity (the default in the PostgreSQL server is 0). 30
Keepalive The number of seconds of connection inactivity before Npgsql sends a keepalive query. 0 (disabled)

Example: Host=myserver.com;Port=5432;Database=mydatabase;User ID=myusername;Password=mypassword;Maximum Pool Size=20;Connection Idle Lifetime=120;Connection Lifetime=180;Keepalive=15;
