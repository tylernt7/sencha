Design Patterns

Diagrams

Creational patterns create classes/objects: how classes are instantiated without the new keyword
  • Singleton pattern shares common data among objects
Structural patterns compose classes/objects: how elements (classes/objects) are combined to form larger structures
  • Composite pattern defines a tree like structure, it focuses on the relationship
Behavioral patterns communicate between classes/objects: how classes/objects interact with each other and how to assign responsibilities to them
  • Observer pattern notifies (and updates) the objects of any changes, it focuses on the communication
