Design Patterns

Diagrams

Creational patterns create objects
  • Singleton pattern shares common data among objects
Structural patterns compose objects
  • Composite pattern defines a tree like structure, it focuses on the relationship
Behavioral patterns communicate between objects: how objects interact with each other
  • Observer pattern notifies objects of any changes, it focuses on the communication
