NetBeans

  • Create a Jave project: File > New Project: Java with Ant, Java Application
  • Generate test file: Select java source file to test, Tools > Create/Update Tests
  • Note: Only JUnit4 Framework is supported with Ant

  • assertTrue
  • assertFalse
