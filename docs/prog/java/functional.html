Functional Interfaces

A functional interface is any interface that contains only one abstract method. A functional interface may contain one or more default methods or static methods. Because a functional interface contains only one abstract method, you can omit the name of that method when you implement it. To do this, instead of using an anonymous class expression, you use a lambda expression.

Lambda Expression Syntax

() -> expression

(parameter) -> expression

(parameter1, parameter2) -> expression

(parameter1, parameter2) -> { code block }

Lambda Expression Desugared DEX Code 
@FunctionalInterface
public interface Square {
    public abstract int calculate(int a);
}

public void calculateSquare() {
    Square square = (int a) -> a * a;

    System.out.println(square.calculate(5)); // 25
}
        
    @FunctionalInterface
    public interface Square {
        public abstract int calculate(int a);
    }

    public class MySquare implements Square {
        @Override
        public int calculate(int a) {
            return a * a;
        }
    }

    private void calculateSquare() {
        System.out.println(new MySquare().calculate(5)); // 25
    }

    // Or using anonymous class
    public void calculateSquareUsingAnonymousClass() {
        Square square = new Square() {
            @Override
            public int calculate(int a) {
                return a * a;
            }
        };
		
        System.out.println(square.calculate(5)); // 25
    }
