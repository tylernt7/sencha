.NET

.NET is a platform/framework, while C# is a (statically typed and object-oriented) programming language that runs on .NET.

.NET (SDK) Components

  • Runtime: Common Language Runtime (CLR)
  • Libraries: Base Class Library (BCL)*
  • Tools: dotnet.exe Command Line Interface (CLI)
* Console.WriteLine is a method in the Console class, which belongs to the System namespace and is available to all .NET applications.

Common .NET SDK CLI Commands

  • build: Build a .NET project.
  • new: Create a new .NET project or file.
  • publish: Publish a .NET project for deployment.
  • run: Build and run a .NET project output.
  • test: Run unit tests using the test runner specified in a .NET project.
  • tool: Install or manage tools that extend the .NET experience.
  • vstest: Run Microsoft Test Engine (VSTest) commands.

.NET Tools

Using dotnet tool install <packageId> --global the following tools can be installed in the user folder:
  • dotnet-counters
  • dotnet-trace
  • dotnet-dump
  • dotnet-ef

.NET Languages

  • C#
  • F#
  • VB.NET (Visual Basic)

TPS (TBD)

Requests per second (RPS) is a measure of how many requests a system can handle in one second, while transactions per second (TPS) is a measure of how many transactions a system can complete in one second. A transaction is a logical unit of work that may consist of one or more requests. For example, a transaction may involve sending a request to a database, receiving a response, and updating the user interface.
