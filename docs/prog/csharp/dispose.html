Dispose method

Implementing the Dispose method is primarily for releasing unmanaged resources. The pattern for disposing an object (dispose pattern), imposes order on the lifetime of an object. The dispose pattern is used for objects that implement the IDisposable interface

  • Managed resources: Objects that can rely on the .NET garbage collector to handle memory management, such as arrays
  • Unmanaged resources: Objects not handled by the .NET garbage collector, such as files, windows, network connections, COM objects or database connections
