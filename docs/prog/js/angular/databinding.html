Types of data binding

Angular provides three categories of data binding according to the direction of data flow:

  • Use [] to bind from source to view
  • Use () to bind from view to source
  • Use [()] to bind in a two-way sequence of view to source to view

One-way binding (Controller to View)

View 
<!-- component.html -->
<h1>{{ title }}</h1> <!-- Interpolation -->
<h1 [innerText]="title"></h1> <!-- Property Binding -->
Controller 
// component.ts
export class AppComponent {
  title = 'Hello World';
}

One-way binding (View to Controller)

View 
<!-- component.html -->		
<h1>{{ title }}</h1>	
<button (click)="onSave()">Save</button> <!-- Event Binding -->
Controller 
// component.ts
export class AppComponent {
  title = 'Hello World';

  onSave() {
    this.title = 'Saved!';
  }
}

Two-way binding (View to Controller to View)

View 
<!-- component.html --> 
<h1>{{ title }}</h1>	
<input [(ngModel)]="title" placeholder="Enter the new title"> <!-- Two-way binding with input field -->
Controller 
// component.ts
export class AppComponent {
  title = 'Hello World';
}
