HTTP Server

Running HTTP Server

Prerequisite

npm install -g http-server

Navigate to the project directory and build the project

ng build

Navigate to the dist/project-name/browser directory and start the server

http-server

Your application should now be accessible at http://localhost:8080

HTTP Server is different from NGINX

For local development and quickly serving static files, http-server is a convenient choice. For more complex and production-ready setups, nginx is a better option.

Source Maps

In web applications, source files are minified and combined into a single main.js file. The code is obfuscated to protect against reverse engineering and to optimize performance. Additionally, source maps are generated during the build process, enabling developers to debug the original TypeScript code while the browser executes the optimized JavaScript.
