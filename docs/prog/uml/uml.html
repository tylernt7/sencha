UML

The Unified Modeling Language (UML) is a general-purpose modeling language that is intended to provide a standard way to visualize the design of a system.

Diagrams

Structure diagrams describe the static aspect (architecture) of the system: things that must be present in the system
  • Class diagram
  • Component diagram
Behavior diagrams describe the dynamic aspect (functionality) of the system: what must happen in the system
  • Use case diagram
  • Sequence diagram

Object Relationships

A bicycle may belong to an owner. A bicycle must own a set of frame parts (strong "has-a" relationship). A bicycle may own one or two wheels (weak "has-a" relationship).

If a bicycle is manufactured, all of its frame parts are manufactured with it. The frame parts are the basic structure (skeleton, main body) of the bicycle. Other parts are installed on the frame.
