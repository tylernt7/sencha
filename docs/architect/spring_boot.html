Packaging
  • JAR (Java ARchive): with an embedded server (Tomcat) for a standalone or microservices application
  • WAR (Web ARchive): deploying to a traditional servlet container or integrating into an existing Java EE environment
Building JAR (Spring Boot 3.1.5, NetBeans)
  • Select File -> Import Project -> From ZIP
  • Select Run -> (Clean and) Build Project
  • WebshopApplication.java -> Run File
Building WAR (Spring Boot 2.7.17, NetBeans)
  • Servers -> Start GlassFish Server 5.0 (JDK 1.8)
  • Servers -> View Domain Admin Console (http://localhost:4848/)
  • Chrome -> Deploy an Application: Location: target\webshop-0.0.1-SNAPSHOT.war, Context Root: /
  • Chrome -> http://localhost:8080/hello
