Understanding the Kafka Ecosystem

Confluent

Confluent is a Data Streaming Platform that was founded by the original co-creators of Apache Kafka.

Component packages:

  • Schema Registry: RESTful schema registry service for managing Avro schemas.
  • Avro: Serialization library.
  • Kafka Streams: a client library for building applications and microservices.
  • ksqlDB: Streaming SQL engine for Kafka.
  • Apache Flink: Stream Processing framework.
