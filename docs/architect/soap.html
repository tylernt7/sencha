Web services (SOAP, WSDL)

Terminology

SOAP (Simple Object Access Protocol): XML-based messaging protocol for structured data exchange.

WSDL (Web Services Description Language): XML-based language describing the operations and messages of a web service.

XML (eXtensible Markup Language): A markup language used to store and transport data.

XSD (XML Schema Definition): A schema that defines the structure and elements of XML documents.
