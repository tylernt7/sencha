AWS (Amazon Web Services)

EC2 (Elastic Compute Cloud)

Launch virtual machines (instances) with customizable configurations. You can run Windows, Linux, or macOS and install any software you need. It delivers secure, reliable, scalable, high-performance, and cost-effective infrastructure (processor, storage, networking). It runs in AWS data centers across the globe.

Lambda (Serverless Computing)

Run code without thinking about servers/clusters. Write Lambda functions in any language (Node.js, Python, Go, Java, and more) and use both serverless and container tools, such as AWS SAM or Docker CLI, to build, test, and deploy your functions. It is best for event-driven architecture. It is designed for short-lived compute tasks that do not retain or rely upon state between invocations. Code can run for up to 15 minutes in a single invocation and a single function can use up to 10,240 MB of memory.
