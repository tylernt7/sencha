Commands

Linux command Windows command Description
sudo runas Run commands as the superuser or root
sudo su runas Enter the superuser or root mode
To exit the superuser mode, type exit or press Ctrl+D in the terminal
ls dir List the files and directories in the current directory (sort entries alphabetically)
pwd cd Print the current (working) directory
chmod -R 777 FILE attrib Change the permissions (mode) of files and directories recursively (full access)
chmod -R 755 FILE attrib Change the permission "read", "write", and "executable" to the owner while the group and other users will only have read and execute permissions.
chmod +x FILE attrib Add execute permissions to the file
cp -a /SOURCE /DEST cp, copy, xcopy Copy the SOURCE directory to the DEST directory
export NAME=Value SET variable=string Set an environment variable with the name NAME and the value Value
envsubst < SOURCE_FILE > DEST_FILE Get-Content SOURCE_FILE | ForEach-Object {Expand-String $_} | Set-Content DEST_FILE (PowerShell) Replace the environment variables in SOURCE_FILE and write the output to DEST_FILE
rm FILE del, erase, rmdir, rm Delete a file named FILE
cat (cat /etc/hosts) type Concatenate or print file(s) to the standard output
nano FILE Notepad++ Open a file with text editor
grep -i 'hello world' menu.h main.c
ps -ef | grep -i linux		 findstr Search for pattern in each file or standard input
hostname hostname Print the name of the current host
ping ping Low-level network connectivity test of a host or IP address
apt-get update choco upgrade all Update the list of available packages and their versions from the repositories
(run before installing any packages)
apt-get install iputils-ping (Linux)
apt-get install inetutils-ping (non-Linux)		 choco install Install the ping package
apt-get install -y dotnet-sdk-6.0 choco install Install the .NET SDK 6.0 on Linux
sudo useradd <username>
sudo passwd <username>		 net user <username> <password> /add Create a user with a password
sudo usermod -aG adm,cdrom,sudo,dip,plugdev,lxd <username> net localgroup Administrators <username> /add Add a user to multiple groups, including sudo
top tasklist Monitor the system performance and the running processes (table of processes)
clear cls Clear the (terminal) screen
exit exit Terminate the current shell session or process
