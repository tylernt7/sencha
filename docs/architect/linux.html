Linux

Commands

Usage Description
sudo Run commands as the superuser or root
ls List the files and directories in the current or specified directory
pwd Print the current working directory
cat Concatenate or print files to the standard output
hostname Display or set the hostname of the system
apt-get update Update the list of available packages and their versions from the repositories
apt-get install ping Install the ping package
apt-get install iputils-ping Install the iputils-ping package
ping Low-level network connectivity test of a host or IP address
top Monitor the system performance and the processes running on it
exit Terminate the current shell session or process
