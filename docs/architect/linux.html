Linux

Commands

Usage Description
sudo Allow you to run other commands as a superuser or another user, as specified by the security policy
ls List the files and directories in the current or specified directory
pwd Print the name of the current working directory
cat Concatenate files and prints them to the standard output. It can also be used to create, view, or append files
hostname Display or set the hostname of the system
apt-get update Update the list of available packages and their versions from the repositories configured on the system
apt-get install ping Install the ping package, which provides a tool for testing network connectivity by sending ICMP echo requests and receiving replies
apt-get install iputils-ping Install the iputils-ping package, which provides a more advanced version of the ping tool with additional features such as IPv6 support, statistics, and flood ping
ping Send ICMP echo requests to a host or IP address and displays the response time and packet loss statistics. It can be used to test network connectivity and performance
exit Terminate the current shell session or process
